Intro

This is a Restful Api for creating user , making blog post , and get all post

Example

Run this code in a console or from any site: 

fetch('https://mysterious-basin-20490.herokuapp.com/post/economic')
            .then(response => response.json())
            .then(json => console.log(json))

Resources

it comes with a set of 3 common resources:

/posts get posts
/users get users *** protected
/auth for authentication

Routes

All HTTP methods are supported.

For login specification mail me @ nahid.mbstu.ict@gmail.com
